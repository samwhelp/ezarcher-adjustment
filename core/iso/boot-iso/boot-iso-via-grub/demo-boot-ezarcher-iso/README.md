## Live Account

| Account / Admin  | Value  |
| ---------------- | ------ |
| Username         | `live` |
| Password         | `live` |


run to change password

``` sh
sudo passwd $(whoami)
```


run to remove password

``` sh
sudo passwd -d $(whoami)
```
