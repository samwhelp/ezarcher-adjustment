
# mate-panel-layouts


## Howto


### install

``` sh
$ ./install.sh
```

or

``` sh
$ make install
```


### package-install

``` sh
$ ./package-install.sh
```

or

``` sh
$ make package-install
```


### config-install

``` sh
$ ./config-install.sh
```

or

``` sh
$ make config-install
```

run to install [brisk-menu](https://aur.archlinux.org/packages/brisk-menu)

``` sh
yay -S brisk-menu
```
