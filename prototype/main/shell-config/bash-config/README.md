## Usage


### install

run

``` sh
./install.sh
```

or run

``` sh
make install
```


### package-install

run

``` sh
./package-install.sh
```

or run

``` sh
make package-install
```

``` sh
./asset-install.sh
```

``` sh
make asset-install
```

``` sh
./config-install.sh
```

or run

``` sh
make config-install
```
