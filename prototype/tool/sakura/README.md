
# Sakura


## Config File Path

* [~/.config/sakura/sakura.conf](config/sakura/sakura.conf)


## AUR Package

* [sakura](https://aur.archlinux.org/packages/sakura)


## Howto


### install

``` sh
$ ./install.sh
```

or

``` sh
$ make install
```


### package-install

``` sh
$ ./package-install.sh
```

or

``` sh
$ make package-install
```


### config-install

``` sh
$ ./config-install.sh
```

or

``` sh
$ make config-install
```
